#samba免密码分享设置
在samba.conf文件中添加一下内容,就能实现samba分享文件无密码访问

```	
[Khadas]
    path = /mnt
    browseable = yes 
    guest ok = yes
    valid users = root
    force create mode =0644
    comment = share all
    browseable = yes
    public = yes
```
